# Library Management System

## Project Description

Library Management System is a Python-based application developed to manage books and library members efficiently. The system allows users to add books, register members, borrow and return books, search books, track overdue books, and maintain library records using JSON file handling.

The project demonstrates the practical use of Object-Oriented Programming (OOP), file handling, modular programming, dictionaries, lists, and data persistence.

---

## Features

* Add New Book
* Register New Member
* Borrow Book
* Return Book
* Search Books by Title, Author, or ISBN
* View All Books
* View All Members
* View Overdue Books
* Save & Exit
* Exit Without Saving
* JSON File Handling
* Data Backup Functionality
* Borrow Limit Management
* Input Validation
* Error Handling
